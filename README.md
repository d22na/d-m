Web Development project 
M&D DELICIOUS

M&D DELICIOUS is a recipe website that allows users to explore, submit, view, manage, and contact the website team about recipes from around the world.

Features
- Home page with recommended recipes
- About Us page
- Submit Recipe page
- View Recipes page
- Manage Recipes page
- Contact Us page
- Simple and clean user interface
- Food images and recipe descriptions
